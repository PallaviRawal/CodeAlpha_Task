The Cloud-Based Bus Pass System is a backend application designed to securely generate and manage bus passes in a cloud-ready environment.
The system focuses on data validation, redundancy prevention, pricing accuracy, and scalability, improving reliability over traditional ticketing systems.

Objectives:
Prevent duplicate bus pass generation
Avoid incorrect pricing and invalid route bookings
Ensure ticket uniqueness to prevent loss or theft
Maintain database accuracy and efficiency
Simulate cloud scalability and high-traffic handling

Technologies Used:
Python – Core application logic
MySQL – Cloud-compatible relational database
Flask-ready architecture (backend service oriented)
SHA-256 Hashing – Secure ticket generation
VS Code – Development environment
Git & GitHub – Version control
